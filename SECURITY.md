# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not** open a public issue.

Report it privately by sending an email to **DIGIT-EIRA@ec.europa.eu**. Please include:

- A description of the vulnerability
- Steps to reproduce it
- Any relevant logs, screenshots, or proof-of-concept code

You can expect an acknowledgement within **[X] business days** and a resolution or status update within **[Y] business days**.

## Supported Versions

| Version | Supported |
|---|---|
| 1.0.0 | ✅ Yes |

## Disclosure Policy

Once a fix is available, the vulnerability will be disclosed publicly via the project's issue tracker on Code.Europa.
